# PttAutoLoginPost
PTT自動登入發文(Python)

## 特色
* 自動登入PTT發文，再登出PTT

   
## 使用方法
請將下列程式碼修改為自己的PTT ID 以及 Password

```
user = 'Your PTT ID'
password = 'Your PTT Password'
```

```
$ python PttAuto.py 

```
* [Demo Video]() - Linux 


###執行環境
Ubuntu 12.04

Python 2.7.3



## 執行範例 

``` 
$ python PttAuto.py 
```
預設為在PTT  Test板 發文
